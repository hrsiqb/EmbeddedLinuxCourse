Commands:

	make CROSS_COMPILE=aarch64-none-linux-gnu- : command to generate write application for 							     cross compiling.
	
	make : command to generate write application for native compiling

	make clean : to clear the writer application file and all the .o files

Files:

	writer.c : application c source file.

	writer : executeable application file.
	
	cross-compile.txt : cross compiled writer application file details
